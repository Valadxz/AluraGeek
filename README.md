## Ejecutar localmente

Clonar el repositorio

```bash
  git clone https://github.com/Valadxz/AluraGeek
```

Ir al directorio del proyecto

```bash
  cd AluraGeek
```

Instalar dependencias

```bash
  npm install
```

Iniciar servidor

```bash
  node server.js
```
